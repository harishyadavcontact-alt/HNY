# Vim

Modes
    <-(Command mode)->Isert Mode
    Normal Mode:Esc
        :q - quit
        :q w - Save quit
    Nav
        W-> Words
        B-> Back
        hjkl-> Left, up, down, right
        gg - Top
        



Use : to temporarily switch to Command-Line mode to save (:w) or exit (:q). 
Press a key like i, a, or o to switch to Insert mode for typing text.


Temux Set up
    Ctrl+a | → split editor vertically (right)

    Ctrl+a - → split editor horizontally (down)

    Ctrl+a h/j/k/l → navigate splits (like Vim / tmux)

    Ctrl+a c → new terminal

    Ctrl+a n/p → move between terminals

    Ctrl+a t → toggle terminal visibility

    Ctrl+a z → toggle Zen Mode (Unix mode)

    Ctrl+a x → close active editor

    Ctrl+a shift+h/l/k/j → resize pane (optional, feels like tmux resize)

    Ctrl+a f → quick file search
