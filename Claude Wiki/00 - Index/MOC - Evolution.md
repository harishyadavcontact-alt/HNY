# MOC — Evolution

> The foundational operating system of all life. Before you can understand humans, you need to understand what shaped them.

---

## Core Framework

```
Selection (exists in abiotic universe)
  + Heredity (passing information across generations)
  = Adaptation (accumulated traits that improve accumulation of limited stuff)
    → Evolution = Descent with Modification
```
