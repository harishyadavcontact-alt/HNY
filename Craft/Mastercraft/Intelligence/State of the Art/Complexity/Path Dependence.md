- Path Dependence
        1. Path Dependence
            History matters and affects outcomes
            Mathematical representation: f(x1,x2..xn)≠f(xn)
                
                Process(A)≠Process(B)even if Endpoint(A)=Endpoint(B)
                
                The outcome depends on the whole sequence, not just the final state
                
                **Qualitative Path Dependence**: Changes in path create qualitative differences, not just quantitative ones
                
            
            **Hysteresis**: System retains memory of past states, Transitions are irreversible
            
            The journey itself transforms the system in ways that cannot be reduced to numeric measures
