# MOC — Science as Method

> Science is not a credential, a degree, or a consensus. It is a method. And we have largely forgotten how it works.

---

## The Method

```
Observation
  → Hypothesis
    → Test (predictive power)
      → Model (useful, not "true")
        → Consilience (all true narratives must reconcile)
```
